## 1. 安装与启动

### 1.1 环境要求
- **操作系统**：Windows 10/11
- **Python**：3.8 或更高版本（运行本程序需要 Python 环境）
- **浏览器**：Chrome 或 Edge 浏览器（同步数据时必须，系统自带Edge即可）
- **网络**：同步 WPS 云文档时需要内网访问权限

> 💡 Python 下载地址：https://www.python.org/downloads/ （安装时勾选"Add Python to PATH"）

### 1.2 依赖安装
首次运行前，需安装以下 Python 依赖包：

```bash
pip install playwright openpyxl xlrd chardet beautifulsoup4 lxml
```

安装 Playwright 后还需初始化浏览器驱动（用于云文档同步）：

```bash
python -m playwright install chromium
```

> ⚠️ 如果系统已安装 Chrome 或 Edge 浏览器，Playwright 会优先使用系统浏览器，无需额外安装 Chromium。

### 1.3 安装步骤
1. 将本文件夹复制到本地任意目录（路径中避免包含中文和空格）
2. 确认 Python 已安装且依赖包已安装
3. 文件夹内应包含：
   - `server.py` — 主服务程序
   - `项目回款跟踪与管控平台_启动.bat` — 启动脚本（双击运行）
   - `停止服务.bat` — 停止服务脚本
   - `sync_data.bat` — 独立数据同步脚本
   - `用户手册.md` — 本手册
   - `管理员手册.md` — 管理员手册

### 1.4 启动方式

**方式一：双击启动脚本（推荐）**
双击 **`项目回款跟踪与管控平台_启动.bat`**，脚本会自动：
1. 检测 Python 是否安装
2. 清理端口 8080 上的旧进程
3. 后台启动服务（无CMD窗口）
4. 等待 3 秒后自动打开浏览器

**方式二：命令行手动启动**
```bash
# 前台运行（可看到日志输出，Ctrl+C停止）
python server.py

# 后台运行（无CMD窗口）
pythonw server.py
```

启动后浏览器访问：**http://localhost:8080**

程序启动后会在同目录下自动创建以下文件夹：
| 目录 | 说明 | 创建时机 |
|------|------|----------|
| `log/` | 日志文件（保留7天） | 启动时自动创建 |
| `data/` | 业务数据文件 | 馶次同步/导入数据时创建 |
| `yundocs_data/` | 云文档原始数据 | 首次云同步时创建 |

### 1.5 停止服务
关闭浏览器页面不会停止服务。需通过以下方式停止：
- **页面停止按钮**：点击页面右上角红色停止按钮⏹️，确认后服务自动停止
- **双击停止脚本**：双击 `停止服务.bat`
- **命令行停止**：运行 `python server.py --stop`
- **Ctrl+C**：如果使用 `python server.py` 前台运行，直接按 Ctrl+C 停止
- 在系统任务管理器中结束 `pythonw.exe` 或 `python.exe` 进程

> ⚠️ 服务停止后，浏览器中的页面将无法访问。下次使用时重新启动即可。

### 1.6 首次使用引导
首次启动后页面为空白（无数据），需要先获取数据：

**方式一：WPS云文档同步（页面操作）**
1. 点击左侧菜单"数据管理"
2. 在"WPS云文档同步"区域输入WPS云文档网址
3. 点击"开始同步"，等待同步完成
4. 页面自动刷新展示数据

**方式二：离线Excel导入（页面操作）**
1. 点击左侧菜单"数据管理"
2. 在"离线Excel导入"区域点击"选择文件"
3. 选择Excel文件后点击"开始导入"
4. 导入完成后页面自动刷新

**方式三：命令行数据同步**
```bash
# 双击 sync_data.bat 或命令行运行：
python fetch_yundocs_full.py [云文档URL]
python preprocess_data.py
```
> 此方式适用于无需启动Web服务、仅需要更新数据的场景。

> ⚠️ 云同步与离线导入互斥，不可同时使用以避免数据错乱。

### 1.7 常见问题

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 启动脚本提示"未检测到Python" | Python未安装或未加入PATH | 安装Python 3.8+，安装时勾选"Add Python to PATH" |
| 启动后浏览器打开但页面空白 | 服务尚未启动完成 | 等待5秒后刷新页面 |
| 提示端口8080被占用 | 其他程序占用端口 | 关闭占用程序，或重启会自动清理旧进程 |
| 首次打开页面无数据 | 尚未导入数据 | 先同步云文档或导入Excel文件 |
| Python运行报错ModuleNotFoundError | 依赖包未安装 | 运行 `pip install playwright openpyxl xlrd chardet beautifulsoup4 lxml` |
| 同步连接中断 | 浏览器未安装或网络问题 | 确认已安装Chrome/Edge，检查云文档地址和网络 |
| 同步失败：未检测到Chrome浏览器 | 系统未安装Chrome且Edge不可用 | 安装Google Chrome或确认Edge可用后重试 |
| pip安装依赖失败 | 网络或pip配置问题 | 检查pip版本和网络连接，尝试 `pip install --upgrade pip` |

## 2. 功能模块

### 2.1 看板首页
看板首页是平台的核心入口，展示所有回款数据的汇总信息。

#### 顶部汇总卡片
- **项目总数**：所有去重项目的总数量
- **计划回款总金额(万)**：所有回款节点计划回款金额总和
- **已回款总合计(万)**：所有回款节点已回款金额总和
- **待回款总金额(万)**：所有节点待回款金额总和
- **总完成率**：已回款 / 计划回款，≥80%绿色、≥50%橘黄、<50%红色

#### 周期切换（年份/季度）
通过顶部右侧Dock组件切换查看：全部 / 本年度 / 下一年度 / 各季度。
- 年度切换采用累计计算逻辑
- 季度切换采用范围过滤逻辑

#### 金额区间卡片
按三个金额区间（100万以上 / 50-100万 / 50万以下）分别展示：
- 已回款合计(万)、待回款合计(万)
- 回款节点数 / 项目总数、完成率
- 6种节点状态的节点数、已回款、待回款、完成率

点击卡片可查看项目详情，点击状态行可下钻到对应状态列表。

#### 季度待回款
以堆叠条形图展示各季度待回款金额，按三个区间着色。与年份联动切换。

#### 月度待回款
以堆叠条形图展示各月份待回款金额，按三个区间着色。
- 悬浮显示金额明细
- 点击柱状图可下钻查看当月所有区间的项目明细（非仅点击部分）
- 与年份联动切换

#### 服务组回款达成排名
展示各PM L3-1部门的回款达成情况，支持按区间筛选、按已回款金额/达成率排序。与年份联动切换。

#### 延期项目 Top10
展示延期天数最多的10个项目，点击可跳转到对应回款节点。

---

### 2.2 回款台账
统一展示所有金额区间的项目，不按区间分离。此页面不响应年份/季度切换，始终显示全部数据。

- 顶部5个核心指标卡片：项目总数、计划回款总金额(万)、已回款总金额(万)、待回款总金额(万)、完成率
- 6个节点状态卡片（高度为核心卡片的1/2）：加资源可提前、达到回款条件、已提前回款、已全额回款、延期、正常实施中
- 区间摘要卡：三个区间的快速统计（标签：计划回款金额、待回款金额）
- 明细表格：支持搜索、区间筛选、状态筛选
- **行内下钻**：点击项目行展开详情，显示项目全字段和回款节点明细
- **设置展示字段**：下钻详情支持自定义显示列

---

### 2.3 回款日历
回款日历拥有独立的年份切换逻辑，不受全局年份切换影响。

- **双月视图**：同时展示两个月日历
- **状态热力图**：日期格根据节点状态着色，悬浮显示真实节点状态信息
- **筛选器**：PM L3-1部门、项目经理L4部门、项目经理
- **15天内到期回款节点**（左侧/橙色）与**30天内到期回款节点**（右侧/蓝色）左右并排展示

---

### 2.4 项目总览 / 回款节点 / 回款状态 / 风险项目 / 数据质检
通过侧边栏进入各金额区间的详细页面。

#### 项目总览
展示该区间所有项目的列表，支持：
- 按服务组、状态筛选
- 关键词搜索
- **设置展示字段**：自定义显示哪些列（支持全选/取消全选）

#### 回款节点
展示该区间所有回款节点的列表，支持：
- 按状态筛选（6种状态）
- 日期范围选择（日历选择器）
- 关键词搜索
- **设置展示字段**
- 首卡显示"回款节点数/项目总数"统计

#### 回款状态
按6种状态分类展示回款节点：
- 加资源可提前
- 达到回款条件
- 已提前回款
- 已全额回款
- 延期
- 正常实施中

页面拆分为上下两区：上方4个核心指标卡片；下方6个节点状态卡片1行排列。

#### 风险项目
展示三类风险：
- **临近到期节点**：7天内到期且未100%回款
- **可提前但未行动**：具备提前条件但未行动
- **高金额低完成率**：完成率<30%且金额最高

#### 数据完整性检查
检查数据缺失情况，支持按L4部门下拉筛选和导出Excel。

---

### 2.5 项目经理视图
按项目经理汇总展示：
- 负责的项目数、金额、已回款、剩余回款、完成率、延期节点数
- 点击行展开两个板块：
  1. **负责项目信息**（支持设置展示字段）
  2. **延期节点信息**（支持设置展示字段）

---

### 2.6 区间对比
三个金额区间的对比分析：
- 回款达成对比卡片
- 回款进度对比（柱状图）
- 状态分布对比（堆叠柱状图）
- 月度回款趋势对比（折线图）
- 服务组达成率排名（TOP5 / BOTTOM5）

---

### 2.7 数据管理
- **WPS云文档同步**：输入WPS云文档网址，同步最新数据。同步中支持"停止同步"。同步前自动检测浏览器可用性，不可用时给出明确提示；数据提取失败时自动中止
- **离线Excel导入**：选择Excel文件导入，与云同步互斥不可同时使用。导入中支持"停止上传"
- **清空数据**：删除所有业务数据（需二次确认）
- **数据质量总览**：展示各检查项的数量
- **数据缺失检查**：全局检查，涉及三个sheet页，列出所有层级列名并分"通用字段"和"差异性字段"展示

> ⚠️ 云同步与离线导入互斥，不可同时使用以避免数据错乱。

## 3. 操作技巧

### 3.1 通用操作
| 操作 | 说明 |
|------|------|
| 侧边栏导航 | 点击左侧菜单项切换页面 |
| 筛选器 | 各页面提供多种筛选条件（区间、状态、部门等） |
| 搜索框 | 输入关键词快速定位项目 |
| 设置展示字段 | 点击表格上方的"设置"按钮，自定义显示列 |
| 下钻查看 | 点击数据行或卡片展开详细信息 |
| 导出Excel | 部分页面支持导出当前数据为Excel文件 |

### 3.2 看板交互
| 操作 | 说明 |
|------|------|
| 周期切换 | 顶部右侧Dock组件切换年份/季度 |

| 状态行点击 | 点击状态行下钻到对应状态列表 |
| 图表悬浮 | 悬浮图表显示金额明细 |

### 3.3 数据管理操作
| 操作 | 说明 |
|------|------|
| 云文档同步 | 输入WPS云文档网址→点击"开始同步" |
| Excel导入 | 点击"选择文件"→选Excel→点击"开始导入" |
| 停止同步/上传 | 操作进行中可随时点击"停止"按钮 |
| 清空数据 | 点击"清空数据"→确认后删除所有业务数据 |
| 命令行同步 | 双击 `sync_data.bat` 或命令行运行提取和预处理脚本 |
